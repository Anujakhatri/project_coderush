# JavaScript Concepts

## 📖 Introduction
This folder contains a collection of simple, interactive examples that demonstrate core JavaScript concepts. Each file showcases a specific feature of the language, allowing you to see the code in action and experiment with it directly in the browser.

---

## 🧩 Topics Covered
| Concept | File(s) Demonstrating | Description |
|---|---|---|
| **Alerts & Console Logging** | `learn.js` | Shows how to display alerts and log output to the console. |
| **Variables & Arithmetic** | `learn.js` | Declares variables (`let`, `const`) and performs basic arithmetic operations. |
| **Functions** | `learn.js` | Defines a simple function (`greet`) and calls it. |
| **DOM Manipulation** | `game.html` | Updates page content dynamically (e.g., displaying mouse coordinates). |
| **Event Listeners** | `game.html` & `learn.js` | Listens for `click` and `mousemove` events and reacts to user interaction. |
| **Dynamic Typing Effect** | `game.html` | Demonstrates a typing animation using `setTimeout`. |
| **Styling via JavaScript** | `learn.js` | Changes the page background color programmatically. |

---

## 📚 Learning Tips
- **Experiment**: Modify variable values, change the text in the `greet` function, or adjust the arithmetic expression. Refresh the page to see the effect.
- **Console**: Use `console.log()` to inspect values and understand the flow of execution.
- **Events**: Try adding more event listeners (e.g., `keydown`) to see how JavaScript reacts to different user actions.
- **Styling**: Play with `document.body.style` properties to change colors, fonts, or layout dynamically.

Happy coding! 🎈
